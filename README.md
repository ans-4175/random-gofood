# Random GoFood

## Quick Start

_If you want to contribute to this project, please fork this repository._

```
cd random-gofood
git checkout main
yarn
yarn start
```

Open at `localhost:3000`

Look for files in
- `src/server.js`
- `src/Home.js`
- `src/App.js`
- `src/data/`
- `src/api/`

## Why is this for?

To help create app faster for fullstack FE-BE in Javascript
